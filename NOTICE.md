# Notice

HappyMeasure is a Python desktop measurement application inspired by
Keith-IVt, a MATLAB desktop app for Keithley 2400 Series SourceMeter IV
workflows.

Upstream reference:
https://github.com/Xiaolong-6/Keith-IVt

Keith-IVt is distributed under the MIT License. HappyMeasure is not a direct
copy of the MATLAB source tree; it is a Python implementation with its own
package structure, tests, hardware abstraction, and Windows packaging scripts.

Before using HappyMeasure with real hardware, run the simulator and software
validation gates, then follow the hardware validation protocol in
`docs/HARDWARE_VALIDATION_PROTOCOL.md`.
